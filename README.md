# Grafo de Recomendação de Música
Este é um algoritmo para recomendação musical com base em grafos, feito através do Neo4j.

# 🎯 Objetivo
Este projeto tem como objetivo criar um algoritmo para recomendação de música, utilizando a linguagem Cypher e o Neo4j, onde o sistema de filtragem recomenda músicas similares ao artista que o usuário mais ouviu, priorizando o que ele ainda não conhece.
